Mario game!
By: Jason Vallee & Noah Hirsch

Keyboard inputs:
Up/Down/Left/Right: Move Mario (all 4 directions are animated)
I/J/K/L: Move the pivot point around which Mario rotates
V: Toggle whether Mario is visible or not
Z: Decrease Mario's alpha
X: Increase Mario's alpha
A: Decrease Mario's size
S: Increase Mario's size
W: Rotate Mario CW about the pivot point
Q: Rotate Mario CCW about the pivot point
F: Decrease the animation speed
G: Increase the animation speed
